# Pretrained weights

Put Swin Transformer pretrained weights here.

Expected names used by the default configuration:

- `Swin Base.pth` for ImageNet-22K pretrained Swin-Base.
- `Swin Base 1k.pth` for ImageNet-1K pretrained Swin-Base used by Stanford Dogs.

Original Swin Transformer weights can be downloaded from the official Swin Transformer release page:

- `swin_base_patch4_window12_384.pth`
- `swin_base_patch4_window12_384_22k.pth`
