# Code and Data Availability statement

Suggested manuscript text:

```latex
\section*{Code and Data Availability}
The source code, training scripts, evaluation scripts, and configuration files of PAAE are available at \url{https://github.com/XubaozZ/PAAE}. The repository provides instructions for environment setup, data preprocessing, dataset organization, and result reproduction. Pretrained models and evaluation scripts are released to facilitate reproducibility and fair comparison.

The experiments are conducted on the publicly available CUB-200-2011, Stanford Dogs datasets and OxfordFlowers. We follow the official or commonly used train/test splits for these datasets. Due to dataset license restrictions, the original images are not redistributed in our repository. Instead, we provide data preparation instructions and split information to help users reproduce the experimental setting.
